killbill-client-python
======================

Python client library for killbill

Installation
------------

The simplest and recommended way to install killbill-client-python is with pip.
You may install the latest stable release from PyPI with pip:

    > pip install killbill

If you do not have pip, you may use easy_install:

    > easy_install killbill

Alternatively, you may download the source package. Extract it and install using:

    > python setup.py install
